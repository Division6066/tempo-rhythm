Agents should use these files as source references only. Do not paste the
prototype wholesale into production. Extract reusable patterns, tokens,
components, screen structure, copy tone, and interaction intent into the
existing Tempo architecture.

The older unpacked reference at `docs/design/claude-export/design-system/`
is still useful for browsing individual source files. This folder preserves
the exact files Amit provided for the April 26 UI workhorse run.
